# 💰 Sistema de Organização Financeira Pessoal

Um sistema web desenvolvido para ajudar no controle financeiro pessoal, permitindo o acompanhamento de receitas, despesas e investimentos.

---

## 📌 Sobre o Projeto

Este projeto tem como objetivo fornecer uma visão clara da vida financeira do usuário por meio de **gráficos interativos** e um **dashboard intuitivo**.

O usuário poderá cadastrar sua renda mensal e registrar suas despesas, que serão automaticamente organizadas em categorias, permitindo uma análise visual eficiente dos seus hábitos financeiros.


---

## 🚀 Funcionalidades

- Cadastro de **receitas (salário e outras fontes)**
- Registro de **despesas**, com categorias:
  - 🏠 Gastos Fixos
  - 🛒 Gastos Variáveis
  - 📈 Investimentos
- Dashboard com:
  - Resumo financeiro geral
  - Saldo disponível
- 📊 Gráficos interativos:
  - Gráfico de pizza (distribuição dos gastos)
  - Gráfico de barras (planejado vs real)
- Sistema de feedback:
  - Alertas de limite ultrapassado
  - Indicadores de conformidade com a regra 50-30-20
- Filtros por **mês** e **categoria**
- CRUD completo (criar, editar, excluir registros)

---

- Manter o equilíbrio entre consumo e investimento
